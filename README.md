# Artqui - Landing Page

Artqui es un SaaS apoyado en Inteligencia Artificial diseñado especialmente para arquitectos y estudios de arquitectura. Esta landing page presenta los pilares fundamentales del proyecto: formación especializada y servicios de IA delegados (renderizado y automatización).

## Características de la Web

- **Diseño Premium**: Paleta oscura, tipografías modernas (Space Grotesk e Inter), estética minimalista y limpia orientada a arquitectos.
- **Visualizador Interactivo**: Un deslizador "Antes / Después" para comparar bocetos a mano frente a renders finalizados por IA.
- **Optimización y Velocidad**: Desarrollado con HTML5, CSS3 moderno y JS Vainilla sin dependencias pesadas.
- **Formulario de Conversión**: Captura de leads optimizada para servicios y formación.

## Despliegue en Vercel

Este proyecto está preparado para ser alojado en Vercel de forma directa:
1. Conecta este repositorio local a GitHub:
   ```bash
   git remote add origin <tu-url-de-github>
   git push -u origin main
   ```
2. Ve a [Vercel](https://vercel.com) e importa tu repositorio de GitHub.
3. Vercel detectará automáticamente que es un proyecto HTML estático y lo desplegará de inmediato.

## Estructura de Archivos

- `index.html` - Estructura semántica del sitio.
- `styles.css` - Estilos globales, variables CSS y diseño responsive.
- `app.js` - Funcionalidad interactiva del visualizador y formulario.
- `assets/` - Directorio con imágenes y recursos visuales.
